1. Clone the repository:
   ```

   ```
2. Navigate to the project directory:
   ```
   cd personal-website
   ```
3. Install dependencies (if applicable):
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```
5. Open your browser and go to `http://localhost:3000`
